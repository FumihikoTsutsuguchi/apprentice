# RealWorld 「Condit」

## はじめに
RealWorldで作られている「Condit」というMedium.comのクローンサイトのバックエンドの API の仕様を満たす Laravel API を作成しました。
* RealWorldリンク: <https://realworld-docs.netlify.app/docs/intro>
## 実装したエンドポイント
* Create Article
* Get Article
* Update Article
* Delete Article
* Registration
* Authentication
* Favorite Article
* Unfavorite Article
